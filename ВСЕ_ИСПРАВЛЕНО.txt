═══════════════════════════════════════════════════════════════

ШАГИ:

1. Узнайте свой Telegram ID:
   → Напишите @userinfobot
   → Скопируйте число (например: 123456789)

2. Создайте файл .env в папке bot:
   → Путь: C:\Users\nokap\Desktop\bot\.env
   → Содержимое смотрите в файле ПРИМЕР_ENV_ФАЙЛА.txt

3. Вставьте в .env:
   BOT_TOKEN=ваш_токен_от_botfather
   ADMIN_IDS=ваш_telegram_id
   DATABASE_URL=sqlite+aiosqlite:///./student_bot.db
   DEBUG=True

4. Сохраните файл .env

5. Перезапустите бота:
   python main.py

6. Проверьте:
   /admin
