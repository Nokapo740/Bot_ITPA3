ШАГ 1: УЗНАЙТЕ СВОЙ TELEGRAM ID (30 секунд)

2. Найдите бота: @userinfobot
3. Напишите ему: /start или привет
4. Он ответит:

   ╔════════════════╗

═══════════════════════════════════════════════════════════════

ШАГ 2: СОЗДАЙТЕ ФАЙЛ .env (1 минута)
───────────────────────────────────────────────────────────────

ВАЖНО! В папке C:\Users\nokap\Desktop\bot\ должен быть файл .env

Если его НЕТ - создайте его:

1. Правой кнопкой мыши в папке bot
2. Создать → Текстовый документ
3. Назовите его: .env
   (без всяких .txt в конце!)

═══════════════════════════════════════════════════════════════

ШАГ 3: ОТКРОЙТЕ ФАЙЛ .env И ВСТАВЬТЕ (1 минута)
───────────────────────────────────────────────────────────────

Откройте файл .env в Блокноте и вставьте:

────────────────────────────────────────────
BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
DATABASE_URL=sqlite+aiosqlite:///./student_bot.db
ADMIN_IDS=ВАШ_TELEGRAM_ID
DEBUG=True
TIMEZONE=Europe/Moscow
────────────────────────────────────────────

ВАЖНО! ЗАМЕНИТЕ:
- BOT_TOKEN - на токен от @BotFather
- ADMIN_IDS - на ВАШЕ число из шага 1

═══════════════════════════════════════════════════════════════

ПРИМЕР ПРАВИЛЬНОГО .env ФАЙЛА:
───────────────────────────────────────────────────────────────

BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
DATABASE_URL=sqlite+aiosqlite:///./student_bot.db
ADMIN_IDS=123456789
DEBUG=True
TIMEZONE=Europe/Moscow

═══════════════════════════════════════════════════════════════

ШАГ 4: СОХРАНИТЕ ФАЙЛ (5 секунд)
───────────────────────────────────────────────────────────────

Нажмите Ctrl + S (или Файл → Сохранить)

УБЕДИТЕСЬ:
✅ Файл называется .env (а не .env.txt!)
✅ Вы заменили ВАШ_ТОКЕН и ВАШ_ID на реальные
✅ Нет лишних пробелов
✅ Сохранили изменения

═══════════════════════════════════════════════════════════════

ШАГ 5: ПЕРЕЗАПУСТИТЕ БОТА (10 секунд)
───────────────────────────────────────────────────────────────

1. Остановите бота (Ctrl+C в консоли где он запущен)
2. Запустите снова:

   python main.py

═══════════════════════════════════════════════════════════════

───────────────────────────────────────────────────────────────

2. Найдите своего бота

ЧАСТЫЕ ОШИБКИ:
───────────────────────────────────────────────────────────────

❌ "У вас нет прав администратора"
   → Проверьте ADMIN_IDS в .env
   → Перезапустите бота

❌ "Invalid token"
   → Проверьте BOT_TOKEN в .env
   → Получите новый токен у @BotFather

❌ Файл .env не находится
   → Создайте в корне папки bot
   → Убедитесь что называется .env (а не .env.txt)

❌ Кнопки не работают
   → Сначала создайте категорию!
   → Потом добавляйте проекты
