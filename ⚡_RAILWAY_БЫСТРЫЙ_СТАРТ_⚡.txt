═══════════════════════════════════════════════════════════════
  📝 КРАТКАЯ ИНСТРУКЦИЯ
═══════════════════════════════════════════════════════════════

1️⃣  РЕГИСТРАЦИЯ (2 минуты)
   └─ https://railway.app
   └─ Login with GitHub

2️⃣  ЗАГРУЗКА НА GITHUB (5 минут)
   └─ https://github.com → New repository
   └─ Загрузите ВСЕ файлы бота
   └─ ⚠️ НЕ загружайте .env!

3️⃣  ДЕПЛОЙ НА RAILWAY (3 минуты)
   └─ Railway → New Project
   └─ Deploy from GitHub repo
   └─ Выберите ваш репозиторий

4️⃣  НАСТРОЙКА ПЕРЕМЕННЫХ (5 минут)
   └─ Railway → Variables
   └─ Добавьте:
       • BOT_TOKEN=ваш_токен
       • ADMIN_IDS=ваш_id
       • DATABASE_URL=sqlite+aiosqlite:///bot.db
       • ENVIRONMENT=production
       • LOG_LEVEL=INFO

5️⃣  ГОТОВО! ✅
   └─ Railway автоматически запустит бота
   └─ Проверьте в Telegram: /start
