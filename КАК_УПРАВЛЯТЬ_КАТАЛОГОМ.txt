➕ ДОБАВИТЬ ПРОЕКТ:
───────────────────────────────────────────────────────────────

1. Отправьте боту: /admin
2. Нажмите: 📚 Управление каталогом
3. Нажмите: ➕ Добавить новый проект
4. Ответьте на 8 вопросов:
   - Название
   - Описание
   - Категория
   - Тип (диплом/курсовая/презентация/проект)
   - Уровень (базовый/средний/продвинутый)
   - Языки программирования
   - Технологии
   - Цена
5. Подтвердите создание ✅

✅ ГОТОВО! Проект в каталоге!

═══════════════════════════════════════════════════════════════

✏️ РЕДАКТИРОВАТЬ ПРОЕКТ:
───────────────────────────────────────────────────────────────

1. /admin → 📚 Управление каталогом
2. Нажмите: 📋 Все проекты (редактировать/удалить)
3. Выберите проект из списка (нажмите на кнопку)
4. Выберите что изменить:
   ✏️ Название
   📄 Описание
   💰 Цена
   💻 Языки
   🔧 Технологии
5. Введите новое значение
6. ✅ Обновлено!

═══════════════════════════════════════════════════════════════

🗑 УДАЛИТЬ ПРОЕКТ:
───────────────────────────────────────────────────────────────

1. /admin → 📚 Управление каталогом
2. 📋 Все проекты (редактировать/удалить)
3. Выберите проект
4. Нажмите: 🗑 Удалить проект
5. Подтвердите удаление ✅

⚠️ ВНИМАНИЕ: Удаление необратимо!

═══════════════════════════════════════════════════════════════

🔴 СКРЫТЬ ПРОЕКТ (не удаляя):
───────────────────────────────────────────────────────────────

1. Откройте проект для редактирования
2. Нажмите: 🔴 Деактивировать
3. ✅ Проект скрыт от пользователей

Чтобы показать снова:
Нажмите: 🟢 Активировать

═══════════════════════════════════════════════════════════════

📋 СПИСОК ВСЕХ ПРОЕКТОВ:
───────────────────────────────────────────────────────────────

/admin → 📚 Управление каталогом → 📋 Все проекты

Здесь вы увидите:
- Все ваши проекты
- Статистику (просмотры, покупки)
- Кнопки для быстрого редактирования

💡 СОВЕТЫ:

✅ Используйте адекватные цены
